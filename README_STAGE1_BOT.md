# Big Cheese Bot — Stage 1 Update

Что изменено:

1. Добавлены новые сегменты:
   - relocation — переезд и адаптация
   - career — работа и карьера
   - family — семья / ребёнок
   - speaking — разговорная уверенность

2. После выбора сегмента бот теперь отправляет не общий CTA, а текст под конкретную ситуацию.

3. В каждое CTA-сообщение добавлены действия:
   - записаться на урок
   - написать Дарье
   - открыть канал Big Cheese

4. 4-дневная прогревающая цепочка переписана:
   - 24 часа: проблема и узнавание ситуации
   - 48 часов: конкретизация решения
   - 72 часа: доверие к подходу Big Cheese
   - 96 часов: финальный CTA

5. Служебная логика заявок сохранена:
   - заявка Дарье в Telegram
   - запись в Google Sheets через SHEET_WEBHOOK_URL
   - Bot Leads / RU Leads / EN Leads / Dashboard работают через уже настроенный Apps Script

Как внедрить:

1. В GitHub заменить файлы:
   - bot.py
   - messages.py

2. Остальные файлы менять не обязательно, если они уже есть в репозитории.

3. Commit changes.

4. Дождаться redeploy Railway.

5. Проверить:
   - /start
   - выбор RU/EN
   - PDF
   - 4 сегмента
   - кнопка “Записаться на урок”
   - кнопка “Написать Дарье”
   - запись в Bot Leads / RU Leads / EN Leads
   - Dashboard

Важно:

- TEST_MODE=1 — тест в минутах.
- TEST_MODE=0 — продакшн 24/48/72/96 часов.
- Для финального режима после теста поставить TEST_MODE=0 в Railway.
